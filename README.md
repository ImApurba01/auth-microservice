# auth-microservice
Auth Microservice is a backend authentication microservice built with Node.js, Express, TypeScript, JWT, and PostgreSQL

## Features
- User registration & login
- Access token
- Refresh token rotation
- Secure cookie storage (httpOnly, secure, sameSite)
- Session revocation on logout
- Refresh token hashes stored in DB

## Tech Stack
- Node.js + Express
- TypeScript
- PostgreSQL
- JWT (jsonwebtoken)
- Crypto (for hashing refresh tokens)
